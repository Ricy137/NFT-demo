# NFT demo

### css-module ts support

Add this plugin to "typescript.tsserver.pluginPaths" in vscode settings.

```json
{
  "typescript.tsserver.pluginPaths": ["typescript-plugin-css-modules"]
}
```
### Getting started

node Version 16+
```
yarn install
```
```
yarn run dev
```
